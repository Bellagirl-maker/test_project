# 📖 [Front-End Developer Test Project] <a name="about-project"></a>

This project is about building a Notion-like editor that has only the H1 heading component.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

This Project was built with HTML, CSS and JavaScript

### Key Features <a name="key-features"></a>

 It is an app that lets you write anywhere and convert your content into H1. This text editor app has the ability to create H1 text simply by typing / then 1, then typing text, then hitting enter.
